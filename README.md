# Elisha-Fit

Elisha-Fit is a web application built with Angular and TypeScript, designed to help users manage and track their fitness journey.

## Features

- User authentication
- Workout tracking
- Progress visualization
- Responsive design

## Technologies

- Angular
- TypeScript
- JavaScript
- npm

### Prerequisites
